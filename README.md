# HTML Survey Form

A semantic HTML5 survey form with labels, inputs, radio buttons, checkboxes, and accessibility features.

## Features
- Collects name, email, and age
- Dropdown for role selection
- Radio buttons for experience rating
- Checkboxes for services used
- Textarea for additional comments

## How to view
Open `index.html` in your browser to test the form locally.
